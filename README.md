# House Price Prediction
House Price Prediction estimates residential property prices using features such as area, bedrooms, bathrooms, location, and age. The project focuses on building a clean pipeline, robust evaluation, and a simple UI for predictions.

## Problem Definition
- Problem statement: Predict the sale price of a house using tabular features that describe its characteristics and location context.
- Type: Supervised Learning (Regression)
- Target variable: SalePrice
- Evaluation metrics: RÂ² (coefficient of determination), MAE (mean absolute error), RMSE (root mean squared error)
